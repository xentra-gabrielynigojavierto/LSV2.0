- createdAtUtc

- integrations: list of MonitoringStatusResponse
- alerts: list of active MonitoringAlertResponse

- join with MonitoredEntity for name/type

ALERTS:
- read MonitoringAlert
- filter active (resolvedAtUtc == null)

SUMMARY:
- aggregate:
  - system status = worst status across entities
  - integrations = all entities with status
  - alerts = active alerts
