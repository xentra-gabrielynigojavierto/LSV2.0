- providers are stored in a **shared global provider registry**
- networks only store **associations to providers**
- adding providers follows:
  - match existing → associate
  - no match → create → associate

- no tenant-owned provider records exist
- provider duplication is minimized
- association model is correctly enforced
- UX supports search → match → associate flow
