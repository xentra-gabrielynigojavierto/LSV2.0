ICON SYSTEM (REMIX ICON)

Install and use Remix Icon:

Option A (preferred):
- Install via npm:
  npm install remixicon

- Import CSS globally:
  import 'remixicon/fonts/remixicon.css'

Option B:
- Use CDN if project prefers (check repo conventions)

Usage rules:

1. Use `<i class="ri-...">` pattern
2. Size consistency:
   - 16px (text inline)
   - 18–20px (sidebar)
   - 20–24px (top bar / primary actions)

3. Color behavior:
   - default: neutral gray
   - active: navy or orange (depending on context)
   - hover: slightly darker tone
   - NEVER random colors

4. Icon mapping examples (use these consistently):

Top bar:
- Product switcher → `ri-apps-2-line`
- Search → `ri-search-line`
- Notifications → `ri-notification-3-line`
- User menu → `ri-user-3-line`

Sidebar:
- Dashboard → `ri-dashboard-line`
- Providers (CareConnect) → `ri-hospital-line`
- Referrals → `ri-file-list-3-line`
- Appointments → `ri-calendar-event-line`
- Messages/Notes → `ri-chat-3-line`
- Settings → `ri-settings-3-line`

Actions:
- Add → `ri-add-line`
- Edit → `ri-pencil-line`
- Delete → `ri-delete-bin-line`
- Filter → `ri-filter-3-line`
- Map → `ri-map-pin-line`
