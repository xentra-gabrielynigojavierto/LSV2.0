Implement LS-LIENS-UI-011 — Provider Mode (Sell vs Manage Internally).

Goal:
Enable dynamic product behavior based on provider mode sourced from a tenant/org configuration API.

Provider modes:
- sell = marketplace-enabled provider workflow
- manage = internal lien management only

This mode is:
- organization-level
- not per-user
- resolved from tenant/org configuration API
- defaulted to sell if missing, invalid, or unavailable

--------------------------------------------------

--------------------------------------------------
MODE SOURCE DECISION
--------------------------------------------------

Provider mode must come from tenant/org configuration API.

Expected mode values:
- sell
- manage

Preferred contract examples:
1.
{
  "providerMode": "sell"
}

or broader config:
2.
{
  "organizationId": "org-123",
  "productCode": "LIENS",
  "settings": {
    "providerMode": "sell"
  }
}

Fallback rule:
- if providerMode is missing, invalid, or API is unavailable → use sell
