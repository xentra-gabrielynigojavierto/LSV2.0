PRIMARY OBJECTIVE:
Add a customer-facing Support portal UI that allows external customers to view and comment on their own CustomerVisible tickets using the secured customer endpoints created in SUP-TNT-03.

IMPORTANT CONTEXT:
Completed:
- SUP-TNT-01 — ExternalCustomer model
- SUP-TNT-02 — Ticket ownership + visibility
- SUP-TNT-02-01 — migration default correction
- SUP-TNT-03 — Customer RBAC Layer

Existing customer endpoints:
- GET  /support/api/customer/tickets
- GET  /support/api/customer/tickets/{id}
- POST /support/api/customer/tickets/{id}/comments

Customer access rules:
- Customer must have JWT role: ExternalCustomer
- JWT must include tenant_id
- JWT must include external_customer_id
- Customer can only access tickets where:
  - TenantId matches JWT tenant_id
  - ExternalCustomerId matches JWT external_customer_id
  - VisibilityScope = CustomerVisible

The UI must be built against the secured endpoints, but real customer login activation may remain deferred until a later block.
