# OBJECTIVE
Enable report execution results to be exported into:
- CSV
- XLSX
- PDF

---

# REQUIRED IMPLEMENTATION

## A. Create Export DTOs

### ExportReportRequest
- TenantId
- TemplateId
- ProductCode
- OrganizationType
- Format (CSV | XLSX | PDF)
- ParametersJson
- RequestedByUserId
- UseOverride (optional, default true)

### ExportReportResponse
- ExportId
- FileName
- ContentType
- FileSize
- GeneratedAtUtc
- Format
- Status
- FileContent (byte[] or base64)

---

## B. Create Exporter Abstraction

### Interface
`IReportExporter`

### Method
`ExportAsync(TabularResultSet data, ExportContext ctx)`

---

## C. Implement Exporters

### 1. CSV Exporter
- comma delimiter
- include headers
- escape values properly
- UTF-8 encoding

### 2. XLSX Exporter
- use ClosedXML (preferred lightweight lib)
- single worksheet
- header row
- typed cells if easy

### 3. PDF Exporter (basic)
- use QuestPDF or iText7
- simple table layout
- no styling complexity

---

## D. Export Service Layer

Create:
- `IReportExportService`
- `ReportExportService`

### Responsibilities
- validate request
- call execution service
- get TabularResultSet
- call correct exporter
- generate file
- build response
- emit audit events

---

## E. File Naming Convention

{TemplateCode}_{yyyyMMdd_HHmmss}.{ext}

---

## F. Integrate Execution Service

- call `IReportExecutionService`
- reuse validation
- DO NOT duplicate execution logic

---

## G. Guardrails

Implement:
- max rows (reuse execution cap)
- max file size (e.g., 10MB)
- reject unsupported formats

---

## H. Audit Events

Emit:
- report.export.started
- report.export.completed
- report.export.failed

Include metadata:
- format
- row count
- file size

---

## I. API Endpoint

Add:

POST `/api/v1/report-exports`

Return:
- file content
- metadata

---

## J. Error Handling

- invalid format → 400
- execution failure → propagate
- export failure → mark failed
