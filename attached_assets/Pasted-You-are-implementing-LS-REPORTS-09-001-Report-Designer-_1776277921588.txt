- output type

### Arithmetic

- `*`

### String

### Basic functions

- `COALESCE`
- `ROUND`

- `LOWER`
- `CONCAT`

## Rules
- formulas may reference only allowed existing fields

- formulas must not allow arbitrary code execution

- calculated fields must be applied in a safe transformation layer after base query execution

## Required work

- validator for allowed functions/operators/field references
- execution-layer transformation support
