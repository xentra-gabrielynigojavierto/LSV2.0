Objective:
Integrate the `liens` microservice with the v2 Identity/authentication pattern so the service can resolve authenticated request context in a platform-aligned way.

This feature is foundational. It is NOT for business permissions or lien workflows yet.

- Identity
- Documents

- Use v2 Identity only

- Do NOT copy code from v1
- Do NOT implement business rules yet
