STEP 3 — DEFINE INTERACTION DESIGN

Make each banner segment clickable:

Click targets:
- Healthy count
- Degraded count
- Down count
- Alerts count

Behavior:

| Click | Result |
|------|--------|
| Healthy | filter list → only Healthy |
| Degraded | filter list → only Degraded |
| Down | filter list → only Down |
| Alerts | filter list → components with active alerts |
| Clicking same active filter | reset to "All" |

UX rules:
- show active filter visually (highlight)
- do NOT redesign banner
- interaction must feel immediate (no loading)
